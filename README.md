# 3DGenerator - AI 3D Model Generation Platform

A full-stack AI-powered 3D model generation platform built with Nuxt 3 + Vue 3 + TresJS.

## Quick Start

```bash
pnpm install
cp .env.example .env  # Add your API keys
pnpm dev
```

## Environment Variables

```env
NUXT_TRIPO_API_KEY=your_tripo_api_key
NUXT_TRIPO_API_BASE=https://api.tripo3d.ai
```

## Deployment

Optimized for Vercel. Set environment variables in Vercel dashboard and deploy.
